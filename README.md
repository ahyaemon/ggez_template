## やっていること

  - 画像の描画

  - キー入力
    - からの移動

  - BGM, SE 再生


## 動かし方
```
git clone
```

```
cargo run
```

デバッグビルドだと bgm がブツ切れになったりするので、ちゃんと動かすときは
```
cargo run --release
```
でリリースビルドする。
